Date Field

The input type="date" is used for input fields that should contain a date.

Input Type Datetime-local

Datetime-local Field

The input type="datetime-local" is used for input fields that should contain a date and time (no time zone).

Note: type="datetime-local" is not supported in Internet Explorer 11 or Safari 12 (or earlier).
